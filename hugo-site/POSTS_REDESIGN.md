# Posts 页面重新设计 - 问题解决方案

## 原始问题

在 `http://localhost:1313/posts/` 页面中：
- 文章列表在分类下面显示
- 滚动时会出现覆盖问题
- 布局简单，缺乏视觉层次

## 解决方案

已将 `/posts/` 页面重新设计为与 `/archive/` 相同的现代风格：

### 核心改进

1. **粘性年份标题**
   - 年份标签固定在顶部
   - 滚动时不会被文章列表覆盖
   - 带有金色左边框和深色渐变背景

2. **时间轴布局**
   - 左侧垂直时间线（金色→粉色→紫色渐变）
   - 每篇文章有圆形节点标记
   - 清晰的时间流动感

3. **卡片式文章列表**
   - 每篇文章独立卡片，带毛玻璃效果
   - 悬停时卡片向右平移
   - 时间节点放大并发光
   - 显示箭头指示

4. **深色学术风格**
   - 深蓝灰渐变背景
   - 金色强调色
   - 优雅字体组合

### 技术实现

修改了 `/hugo-site/themes/custom-theme/layouts/_default/list.html`：
- 复用 `archive.css` 样式
- 自动从文件名提取日期（格式：YYYY-MM-DD-title.md）
- 按年份自动分组
- 显示分类标签

### 访问方式

1. **开发服务器**：
   - `http://localhost:1313/posts/`
   - `http://172.24.230.68:1313/posts/`

2. **静态文件**：
   - `hugo-site/public/posts/index.html`

### 响应式支持

- **桌面端**：完整时间轴布局
- **平板**：调整间距和字体
- **手机**：简化布局，日期换行

### 统一体验

现在 `/posts/` 和 `/archive/` 两个页面使用相同的设计风格，提供一致的用户体验。

## 构建和部署

```bash
cd hugo-site
hugo --cleanDestinationDir
hugo server
```

访问 `http://localhost:1313/posts/` 查看新设计！
