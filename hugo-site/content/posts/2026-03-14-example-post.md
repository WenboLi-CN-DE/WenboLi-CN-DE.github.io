# 示例文章

这是一篇示例文章，用于展示博客的新设计。

## 卡片元素

文章卡片包含以下元素：

1. 封面图（可选）
2. 分类标签
3. 阅读时间
4. 文章标题
5. 文章摘要
6. 发布日期
7. 作者信息

## 设计特点

- 卡片式布局
- 响应式网格
- hover 动画效果
- 懒加载图片
- 暗色模式支持

```yaml

```

这样就可以显示完整的卡片效果了。
